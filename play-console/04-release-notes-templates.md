# Release Notes Templates

Use these templates in the Play Console release notes field.

## Template A: Initial release

Oh Craps! is now available on Android.

This first release includes:

- Strategies tab with searchable and filterable strategy list
- Rules tab with accessible expandable sections
- Create Strategy tab with fully accessible form and saved strategy management
- About tab with references, credits, and feedback action
- Native Android sharing for core and user-created strategies
- TalkBack-focused navigation and action support across main flows

## Template B: Initial release short version

Initial Android release of Oh Craps!

## Recommended style guidance

- Keep first line focused on the highest-impact change
- Use plain language and short bullets
- Avoid internal terms like slice numbers or ticket IDs
