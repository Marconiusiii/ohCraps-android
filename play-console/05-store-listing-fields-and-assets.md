# Store Listing Fields And Assets

Use this as a prep worksheet before filling the Play Console listing.

Date baseline: February 16, 2026.

## Core listing fields

- [ ] App name
- Suggested: `Oh Craps!`

- [ ] Short description
- Suggested draft:
- `Accessible craps strategy reference app with searchable systems, rules, and strategy creation tools.`

- [ ] Full description
- Suggested draft:
- `Oh Craps! brings strategy-focused craps reference tools to Android with accessibility-first design. Browse and search a large strategy library, filter by table minimum and buy-in, read detailed strategy steps, and create your own custom strategies. Save, edit, duplicate, delete, submit, and share strategies using native Android workflows. The app includes a Rules section, references, and built-in feedback options.`

- [ ] App category
- Suggested: `Education` or `Tools` (choose what best matches your positioning)

- [ ] Contact email
- [ ] Website URL
- [ ] Privacy policy URL

## Graphic assets checklist

- [ ] App icon (512x512)
- [ ] Feature graphic (1024x500)
- [ ] Phone screenshots
- [ ] 7-inch or 10-inch tablet screenshots if you plan tablet distribution

Asset source guidance:

- Prepare clear screenshots showing each tab and key workflows.
- Use readable text size and visible controls for accessibility-forward presentation.

## Suggested screenshot set

- [ ] Strategies list with search and filters visible
- [ ] Strategy detail screen
- [ ] Rules expandable sections
- [ ] Create Strategy form
- [ ] My Strategies list with actions
- [ ] About screen with references

## Localization planning

- [ ] Default language listing complete
- [ ] Additional locales planned or deferred

## Policy and review readiness checks

- [ ] Listing copy does not claim unsupported features
- [ ] Listing copy matches current behavior
- [ ] Privacy policy URL is live and correct
- [ ] Data safety answers are aligned with real app behavior
